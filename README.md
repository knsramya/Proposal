**Proposal** is a fun, interactive single-page quiz that asks "Do you love me?" using **Studio Ghibli-style portraits** generated from real photos.  
It includes playful animations: moving "No" button, heart emojis, and floating petals! Perfect for sharing or creating a personalized web experience.

## Features

- Single-page app with multiple interactive screens
- "No" button moves away on hover for playful interaction
- Hearts animation and floating petals on the final screen
- Fully responsive design (mobile-friendly)
- Replace GIFs with custom Studio Ghibli portraits generated from your photos

## Tech Stack

- HTML, CSS, JavaScript
- Tenor embed for GIFs (or your own Ghibli images)
- Fully client-side, no backend required

## Setup / Usage

1. Clone the repo:
```bash
git clone https://github.com/Sama-ndari/Proposal.git
````

2. Open `index.html` in your browser.
3. Optional: Replace GIFs with Ghibli-style images generated from your selfies.
4. Host on GitHub Pages or any static hosting service.
